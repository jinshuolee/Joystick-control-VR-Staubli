# 摘要
本專案以Unity做為開發平台，搭配Meta Oculus Quest 2的VR頭盔和手把進行Staubli實體機械手臂的控制，使用者可以透過操作Oculus的手把移動Joystick的末端點，透過程式計算每個關節的角度，達到同步控制Staubli 實體機械手臂的功能。
